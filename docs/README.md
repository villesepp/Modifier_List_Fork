# Modifier List Fork

<img width="684" height="615" alt="image" src="https://github.com/user-attachments/assets/721cc00e-90f7-450b-97c6-d9deff0d3e54" />

This is a vibe-fixed fork of the handy Modifier List add-on. (https://extensions.blender.org/add-ons/modifier-list-fork/)

__Shortly put, it's a massive time-saver if you enjoy using modifiers in Blender. It gives you a quicker access to modifiers by a simple press of alt+space.__

This fork
- restores the functionality to add a modifier using the search that is visible immediately (as opposed to the one that is behind a click of the "+" button).
- gives focus to the search option so you can immediately start search and add a modifier

Briefly tested using Blender 5.1.2, and seems to work.
